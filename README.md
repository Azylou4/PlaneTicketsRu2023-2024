
# Анализ данных дешевых авиабилетов в РФ 2023-2024
Анализ стоимости виабилетов в России 2023-2024 с ипользованием дата-скрейпинга из Телеграм-канала.
Цель проекта — выявить наиболее выгодные маршруты и города для путешествий на основе исторических данных о ценах.

## Описание

Проект включает в себя:
- Сбор данных из телеграм-канала с помощью парсера.
- Обработка и очистка данных.
- Анализ цен на билеты по маршрутам и городам.
- Визуализация результатов с помощью графиков.


## Анализ данных

Проект анализирует следующие аспекты:
- Средняя цена билетов по маршрутам.
- Сравнение цен на билеты из разных городов.

После выполнения анализа получаем таблицы и графики, отображающие:
- Средние цены на билеты по маршрутам
- Город с наиболее дешевыми билетами на вылет
